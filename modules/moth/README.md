# Moth, a lua device event listener
Influenced from: https://github.com/stefano-m

## Linux (Mainly D-Bus)
 - Pipewire (Future)
 - Pulseaudio
 - Alsa (fallback)
 - Upower
 - Udisk (fork udiskie for lua)
 - Display/Randr/Redshift/geoclue/ display color
 - Network Manager (https://developer.gnome.org/NetworkManager/stable/spec.html)
 - Bluetooth
 - Task Warrior
 - Ledger

## Other (non-dbus)
 - Just use process monitor (much more information)
 - Not needed for daily use
 - CPU
 - MEM
 - Network up/down
 - tp_smapi
